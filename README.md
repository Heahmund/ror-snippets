# ror-snippets README

Ускорьте разработку на Ruby on Rails с помощью полезных сниппетов прямо в VS Code!
Это расширение добавляет часто используемые шаблоны кода, чтобы вы могли сосредоточиться на логике приложения.

Начните с удобного  **одноразового дебаггера** , и следите за обновлениями — в будущем появятся новые сниппеты для Rails и Ruby.

## Features

### One-time Debugger

Рассчитан на наличие gem debug
Иногда вы хотите, чтобы отладчик сработал  **только один раз** , особенно в циклах или часто вызываемых методах. Этот сниппет вставляет идиоматичную Ruby-строку, гарантирующую однократное срабатывание:

> @debugger_once ||= (debugger; true)

* Prefix: `odebug`
* Описание: Insert one-time debugger line

Просто введите `odebug` в `.rb` файле и нажмите `Tab` — сниппет вставится автоматически.
